# DBのテーブル設計、マイグレーション

## ✅ 行ったこと

- DBのテーブル設計
- マイグレーション

## テーブル設計

| カラム名 | 内容 |
| ---- | ---- |
| title | ゲーム名 |
| platform | プラットフォーム |
| rating | 5段階評価 |
| review | 感想 |
| price | 値段 |
| condition | 新品か中古か |
| status | 未プレイ / プレイ中 / クリア済み |
| play_time_minutes | プレイ時間 |

## 実装

下記コマンドを実行してマイグレーションファイルを作成
```
php artisan make:model Game -m
```
コードの意味：**Gameモデルと、gamesテーブルを作るためのマイグレーションファイルを同時に作成するコマンド**
