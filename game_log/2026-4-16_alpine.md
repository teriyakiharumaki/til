## 一覧画面にレビューを折りたたみ表示する
game_log/resources/views/games/index.blade.php
```
        @method('DELETE')
        <button type="submit" onclick="return confirm('この記録を削除しますか？')">削除</button>
      </form>

      <div x-data="{ open: false }">                                            #ここの部分
        <button type="button" @click="open = !open" style="margin:6px;">
          <span x-text="open ? 'レビューを閉じる' : 'レビューを見る'"></span>
        </button>

        <div x-show="open" x-transition style="margin:6px;">
          {{ $game->review }}
        </div>
      </div>
    </li>
  @endforeach
</ul>
```

### ①`x-data="{ open: false }"`

**`open: false` って何？**
- open = 開いてるかどうかを表す名前
- false = 最初は閉じてる

つまり最初の状態は、レビューは閉じている

たとえると
```
open = スイッチ
false = OFF
true = ON
```

### ② `@click="open = !open"`

ボタンをクリックした時に、
```
open = !open
```
を実行する

**`!open` って何？**

! は「反転」だから
```
open が false なら → true
open が true なら → false
```
になる、つまり

ボタンを押すたびに
```
閉じる → 開く → 閉じる → 開く
```
と切り替わる

### ③`x-text="open ? 'レビューを閉じる' : 'レビューを見る'"`

**open の状態に応じて、表示する文字を変える**

? : の意味は、三項演算子
```
条件 ? 条件がtrueのとき : 条件がfalseのとき
```

今回だと
```
open ? 'レビューを閉じる' : 'レビューを見る'
```
なので
```
open = true → 「レビューを閉じる」
open = false → 「レビューを見る」
```
になる

### ④`x-show="open"`

**open が true のときだけ、div を表示**

- open = false → 非表示
- open = true → 表示

レビュー本文の表示/非表示を制御している

### ⑤`x-transition`

**表示・非表示が切り替わる時に、ふわっとしたアニメーションを付ける**

### ⑥`type="button"`

```
<button type="button">
```

**なぜ必要？**

もしこのボタンが `<form>` の中にあると、
button はデフォルトで submitボタン扱い になることがある

でも今回は送信したいわけじゃないので、
```
type="button"
```
を付けて、「ただのボタンです」と明示している
