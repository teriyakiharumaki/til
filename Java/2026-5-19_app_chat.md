### Websocketの追加

参考サイトではSpring Initializrで依存関係を入れていたが、今回は既存のアプリに導入

**Spring Initializrは「最初にpom.xmlへ依存関係を書いてくれる道具」なので、あとから手で足すことができる**

なので、既存のpom.xml の `<dependencies>` の中に、これを追加する

```
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-websocket</artifactId>
</dependency>
```

Spring Boot公式でも、WebSocket機能は spring-boot-starter-websocket で利用できると説明されている<br>
(https://docs.spring.io/spring-boot/reference/messaging/websockets.html?utm_source=chatgpt.com)
